Detailed Description

Controller of Xapian::MatchSet.

This class do a lot of stuff. User can extend it for working other variants of MatchSpy.

getIteratorGenerator allows to create an object of type TermIteratorGenerator, which can be used inside TermQlcTable.

If the MatchSet cannot be used as TermGenerator, do not overwrite these methods. If the Erlang user will try to use this MatchSpy as a term iterator, an exception will be thrown.

MatchSpy can filled only once, after that it will be finalize.

Objects of this type are reference-counted. MatchSpy will be deleted in the destructor.
